Coleccion de Dinosaurios

esta es una coleccion donde encontraremos gran cantidad de los dinosaurios que podemos observar al estar por el mundo de ARk, nuestro survival favorito de confianza

  • Traer todos los dinos /dinosaurios
  • Traer un dino en especifico /dinosaurios/Argentavis
  • Borra un dino especifico si mandamos esta url con un metodo "delete" /dinosaurios/Argentavis
  • Con estas url podemos obtener dinos en base a su modo de alimentacion "dinosaurios/nutricion/Herbivoro-Carnivoro-piscivoro" Herbivoros, Carnivorós, piscivoros
  • Con esta url podemos obtener una cantidad especifica de dinos /dinosaurios/cantidad/5
  • Con este link podemos ordenar alfabeticamente nuestros dinosaurios /dinosaurios/orden/mayor
  • Creamos un dino mandandolo por medio de post especificando los datos ejemplo: /dinosaurios 
        "nombre": "Argentavis",
    "descripcion": "Dominando los cielos de toda la isla",
    "doma": "si",
    "nutricion": "Carnivoró",
    "comportamiento": "Irritable",
    "imagen": "./img/Argentavis.jpg",
    "vida": "86"
  • Con esta url enviandola con un metodo put puedes modificar un dinosaurio existente en la base de datos /dinosaurios

Coleccion de Mapas

esta es una coleccion donde encontraremos gran cantidad de los mapas que podemos observar al estar por el mundo de ARk, nuestro survival favorito de confianza

  • Traer todos los Mapas /mapas
  • Traer un Mapa en especifico /mapas/Scorched_Earth
  • Borra un Mapa especifico si mandamos esta url con un metodo "DELETE" /mapas/Scorched_Earth
  • Con esta url podemos obtener una cantidad especifica de Mapas /mapas/cantidad/5
  • Con este link podemos ordenar alfabeticamente nuestros Mapas /mapas/orden/mayor
  • Creamos un mapa mandandolo por medio de "POST" especificando los datos ejemplo: /mapas 
        "nombre": "Scorched_Earth",
    "descripcion": "Scorched Earth un mundo Desertico.",
    "objetos especiales": "si",
    "multijugador": "si"
  • Con esta url enviandola con un metodo "PUT" puedes modificar un mapa existente en la base de datos /mapas
